# Ansible Playbook Creator Plugin

Create Ansible playbooks for configuration management.

## Usage

```bash
/ansible-playbook
```

## Features

- Production-ready playbook generation
- Security scanning with `secure_playbook.py`
- Playbook validation with `validate_playbook.py`
- Container-based testing with `test_playbook.sh`
- Best practices and idempotency rules
- Multi-platform support
