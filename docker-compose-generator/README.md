# Docker/Podman Compose Generator Plugin

Generate production-ready Docker Compose and Podman Compose configurations.

## Usage

```bash
/docker-compose
```

## Features

- Multi-service orchestration
- Health check configurations
- Volume and network management
- Resource limits
- Environment variable support
- Production best practices
- Podman Compose compatibility (rootless, SELinux, pod-based networking)
