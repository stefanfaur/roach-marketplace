# Writing Natural Language

## Mandatory First Step

**You MUST read the full contents of `${CLAUDE_PLUGIN_ROOT}/lib/elements-of-style.md` using the Read tool before writing a single word. Do not proceed until you have done so.**

This is not optional. Read the file. Then write.

## Apply While Writing

Focus on these rules from the Elements of Style:

- **Rule 10**: Use the active voice
- **Rule 11**: Put statements in positive form
- **Rule 12**: Use definite, specific, concrete language
- **Rule 13**: Omit needless words
- **Rule 16**: Keep related words together
- **Rule 18**: Place the emphatic words of a sentence at the end

## That's It

No modes. No capture. No finalize. No subagents. No file organization.

Read the style guide. Write. Apply the rules.
